Lesson 3 Classwork Task 1

  1. Массивы в JS являются объектами, поэтому и сравниваются как объекты.
    [] == [] -> false, поскольку внутренние ссылки обоих объектов, ссылаются на разные объекты в памяти.
  2. В логическом контексте пустой массив - true.
    Операция ![] дает результат false.
  3. При сравнении объекта [] с примитивом он также пробразуется в примитив с помощью метода toString().
    [].toString() -> пустая строка -> false
  4. false == false -> true
